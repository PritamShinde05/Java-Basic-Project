# 🧩 Level 1 – Core Java Fundamentals

## 📖 Overview

This level focuses on building a strong foundation in Java by practicing core programming concepts such as syntax, arrays, loops, conditionals, and user input handling.

All programs in this level are designed to strengthen logical thinking and problem-solving skills through simple yet practical implementations.

---

## 🛠️ Tasks Implemented

### 🔹 Task 1: 3×3 Number Grid & Search

* Created a 3×3 grid using arrays
* Displayed the grid in matrix format
* Implemented logic to check whether a given number exists in the grid

👉 **Concepts Used:** Arrays, Nested Loops, Conditionals

---

### 🔹 Task 2: Temperature Converter

* Converts temperature between **Celsius and Fahrenheit**
* Takes user input and performs dynamic conversion

👉 **Concepts Used:** Formulas, User Input, Conditional Logic

---

### 🔹 Task 3: Student Grade Average Calculator

* Accepts marks of multiple students using arrays
* Calculates total and average marks
* Displays results clearly

👉 **Concepts Used:** Arrays, Loops, Arithmetic Operations

---

### 🔹 Task 4: Secure Password Generator

* Generates random passwords based on user-defined criteria
* Includes uppercase, lowercase, digits, and special characters

👉 **Concepts Used:** Random Class, Strings, Loops, Conditional Logic

---

## 🎯 Skills Developed

* Java Syntax & Structure
* Arrays and Iteration
* Conditional Statements
* User Input Handling (Scanner)
* Basic Problem Solving

---

## ▶️ How to Run

1. Compile the Java file:

```
javac FileName.java
```

2. Run the program:

```
java FileName
```

---

## 📌 Notes

* Each task is implemented as a separate Java file
* Code is written with clarity and proper structure
* Focus is on understanding logic rather than optimization

## 🚀 Outcome

Successfully built a strong base in Java fundamentals, preparing for more advanced concepts like file handling, OOP, and real-world applications in upcoming levels.
