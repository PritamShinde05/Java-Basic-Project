# 🚀 Java Practice Project – From Basics to Real-World Applications

This repository showcases my structured journey of learning Java, progressing from core fundamentals to building real-world applications.

The project is divided into **5 levels**, each focusing on specific concepts and practical implementation.

---

## 📌 Project Structure

* **Level 1** – Core Java Fundamentals 
* **Level 2** – Intermediate Java Utilities 
* **Level 3** – OOP & Small Projects
* **Level 4** – Real-World Beginner Projects
* **Level 5** – Advanced Beginner / Professional Projects

---

# 🧩 Level 1 – Core Java Fundamentals

### 📖 Description

This level builds a strong foundation in Java syntax and logic using arrays, conditionals, loops, and user input.

### 🛠️ Tasks

* Display a **3×3 number grid** using arrays and check if a number exists
* Create a **Temperature Converter** (Celsius ↔ Fahrenheit)
* Calculate **Student Grade Averages** using arrays
* Generate a **Secure Random Password** based on user preferences

### 🎯 Skills Gained

* Java Basics
* Arrays & Loops
* Conditional Statements
* User Input Handling

---
